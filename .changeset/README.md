# Changesets

Hallo und Willkommen! Dieser Ordner wurde automatisch von `@changesets/cli` generiert, einem Build-Tool, das
mit Multi-Package-Repos oder Single-Package-Repos arbeitet, um Ihnen bei der Versionierung und Veröffentlichung Ihres Codes zu helfen. Sie können
die vollständige Dokumentation dafür [in unserem Repository](https://github.com/changesets/changesets) finden.

Wir haben eine kurze Liste häufig gestellter Fragen, um Ihnen den Einstieg in dieses Projekt zu erleichtern
[in unserer Dokumentation](https://github.com/changesets/changesets/blob/main/docs/common-questions.md)
